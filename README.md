# notes-app-laravel-react

Aplicativo de notas feito com Laravel e React para treinar minhas skills.

## Ferramentas utilizadas

- Laravel (API)
- React (Cliente)
  - Redux
  - Redux Thunk
  - React-Redux
  - Hooks

## O que deve haver

- Uma página inicial com:
  - Uma lista de notas
  - Um botão para adicionar uma nota
